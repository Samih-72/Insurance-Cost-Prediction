
# Insurance Cost Prediction using Machine Learning

## Project Overview

This project predicts medical insurance charges based on customer demographics and health-related attributes such as age, BMI, smoking status, region, and number of children.

The goal is to understand which factors influence insurance costs and build a machine learning model capable of estimating future insurance charges.

## Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker Status
* Region
* Insurance Charges (Target Variable)

## Project Workflow

1. Data Collection
2. Exploratory Data Analysis (EDA)
3. Data Cleaning
4. Feature Engineering
5. Feature Scaling
6. Feature Selection
7. Model Building using Linear Regression
8. Model Evaluation using R² Score and Adjusted R²

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* SciPy

## Results

The Linear Regression model was trained to predict insurance charges based on customer information. Model performance was evaluated using R² Score and Adjusted R².
